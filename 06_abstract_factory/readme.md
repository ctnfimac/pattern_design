## Abstract Factory:
Es un patrón de diseño que permite crear familias de objetos relacionados sin especificar sus clases concretas.

En lugar de crear objetos individuales directamente, creamos fábricas que producen un conjunto de objetos relacionados.

Es útil cuando necesitas crear objetos que son parte de una familia y quieres asegurarte de que estos objetos se complementen entre sí.
